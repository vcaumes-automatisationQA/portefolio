# Portfolio de Tests Automatisés

Ce portfolio présente des projets de tests automatisés réalisés avec trois technologies différentes : Playwright, Selenium, et Robot Framework.  
Les tests sont basés sur le site de démonstration [Sauce Demo](https://www.saucedemo.com/), ce qui permet de les exécuter facilement.

## Structure des dossiers
- \saucedemo\playwright : tests automatisés avec Playwright
- \saucedemo\selenium : tests automatisés avec Selenium
- \saucedemo\robotframework : tests automatisés avec Robot Framework

## Instructions
Chaque dossier contient son propre README avec les instructions pour exécuter les tests.
